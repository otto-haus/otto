## Requirements

- Bun
- go-task
- Letta Desktop or a local Letta runtime for live chat work

## Install

```sh
git clone https://github.com/otto-haus/otto.git
cd otto
bun install --frozen-lockfile
```

## First checks

```sh
task ci
```

`task ci` runs the same local gate that GitHub CI runs for pull requests. See
[Local CI gate](/ci-gate) for the current command list.

## Agent extension

```sh
bun run install-extension
```

Then run `/reload` in Letta Code.

Provider credentials belong in Letta, not otto.
