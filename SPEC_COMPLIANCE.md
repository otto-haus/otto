# Otto v0.1 — SPEC_COMPLIANCE.md

This matrix audits each public surface in the Otto v0.1 spec against six evidence
dimensions: **File contract** (spec/types/templates exist), **Runtime** (code that
actually executes the behavior), **Desktop** (the workspace surface in `apps/desktop`),
**Tests** (automated coverage), **Demo** (a video in `demo/out/`), and **Receipt** (a
proof artifact in `receipts/otto-v01/`). Status is the conservative roll-up; Ship
decision honors the truth rules ("no artifact, no progress"; prototype-only ≠ shipped;
re-enactment ≠ live capture; nothing is Shipped until Sebastian signs off).

Legend: **done** = real and verifiable · **partial** = exists but incomplete/unwired ·
**missing** = not present · **na** = not applicable to this surface.
Two gaps from the source audit were re-verified against disk and **downgraded** because
the files now exist: `docs/desktop-convergence.md` (3.7 KB, real content) and
`templates/autonomy-receipt.md` (491 B, real template).

## v0.1 cutline

**Ship (3) — real and verifiable:**
- **namespace** — clean public identity, all dimensions done.
- **practices** — loader/validator/CLI + 6 unit tests + real practices.json on desktop. Only P2s.
- **skills** — full package + docs + install; one P1 (live-agent load not proven) is acceptable for a preview.

**Proposed (4) — conditional ship, honest preview language required:**
- **desktop** — workspace shell typechecks/builds; chat is prototype, clearly labeled. (Disable the chat input.)
- **charter** — gate hook is genuinely live; Auditor + end-to-end captured run are not. Ship as "operating contract + working gates," not "proves done automatically."
- **routines** — specs + one-off trials + correct approval gate; recurring scheduler deferred. Ship as documented canon.
- **standards** — live canon with a real blocking precedent; automated enforcement deferred. Ship as canon, not as an enforced gate.

**Defer (7) — Built, not Shipped:**
- **approvals, autonomy, tickets, knowledge, runs-receipts, worker-orchestration, channels** — each is spec/template/prototype with no closing runtime. Keep the deferred/proposed labels exactly as written.

**Cut (1):**
- **curation** — no schema, no queue, no lifecycle. Remove any implication it exists in v0.1; it is the missing spine that makes most "enforcement" claims aspirational.
