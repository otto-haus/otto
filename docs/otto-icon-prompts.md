# otto icon prompts

## Brand rule

- The owl is the primary app mark and avatar.
- Sidebar/action icons should be custom line icons in the same family — not literal owls everywhere.
- Style: warm paper, black ink, rounded geometry, quiet intelligence, no mascot/cute energy.

## App icon prompt

```txt
Design a premium macOS app icon for "otto": a minimal owl face made from black ink shapes on a warm off-white rounded-square background. Calm, intelligent, restrained, elegant. No text. No letters. No cartoon. No corporate mascot. No busy detail. Use soft geometry, balanced negative space, and subtle paper depth only. The icon should read clearly at 16px, 32px, 128px, and 512px.
```

## Transparent owl mark prompt

```txt
Create a minimalist owl mark for "otto", an AI operating companion. Elegant black ink linework, subtle owl face geometry, rounded proportions, calm and premium. Transparent background. No text. No letters. No logo lockup. No gradients. No cute mascot style. SVG-like flat vector, readable at tiny sizes.
```

## Icon family master prompt

```txt
Create a custom minimalist line icon for the otto desktop app.

Style:
- elegant black ink linework
- warm off-white or transparent background
- rounded corners and soft geometry
- thin-to-medium stroke
- calm, premium, Swiss interface feel
- slight owl-inspired DNA only when natural: paired dots, soft beak notch, feather-like angle
- no text, no letters, no logos
- not cute, not cartoon, not corporate
- readable at 16px, 24px, and 32px

Output:
single centered flat vector icon, SVG-like, monochrome.
```
