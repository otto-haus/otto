## Summary

Adds a first-class, Labs-gated entry point for the receipt → practice-mining → Curation
leg of the north star.

**Finding (honest):** `PracticeMiningLoop.observe()` was implemented and tested but had no
direct invocation path with a proper Labs gate. It was only reachable indirectly via
`routines.runManual('practice-mining')`. The `practice_mining` Labs flag existed but was
display-only.

### Change

- New `triggerPracticeMining()` (`practice-mining-trigger.ts`):
  - Labs gate: `labs.enabled && labs.features?.practice_mining`
  - Runs `observe()` over the receipts dir (receipts only — no conversation, never
    `conversation=default`)
  - Returns explicit `{ gated: true }` marker when Labs is off; nothing fires silently
- Wired to `otto:practices:mine` IPC handler + preload `practices.mine()`
- Drafts Practice proposals that surface in Curation; writes `practice.mining.observe` receipt
