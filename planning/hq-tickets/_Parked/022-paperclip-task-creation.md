# 022 — Paperclip Task Creation

Owner: Cursor
Priority: P2
Depends on: 021

## Outcome

Otto can create Paperclip work while keeping Curation/approval authority.

## Scope

- Create Paperclip goals/tasks from approved Routines or Charters.
- Track worker status.
- Import worker results into Otto receipts.
- Route behavior implications into Curation.

## Done when

- Otto creates a Paperclip task from an approved need.
- Worker completes it.
- Result imports into Otto receipt/Curation.
- Consequential follow-up requires approval.
