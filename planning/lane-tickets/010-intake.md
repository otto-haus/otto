# 010 — Intake

Status: todo
Owner: future
Priority: P1
Depends on: 008, 016

## Outcome

Otto can import thinking surfaces and turn them into curated proposals.

## Scope

- Manual import first.
- Supported inputs: markdown notes, Claude/ChatGPT exports, meeting transcripts.
- Extract decisions, corrections, lessons, objections, claims, and open loops.
- Every extracted item keeps a source link or source excerpt.
- Candidates go to Curation, not directly into canon.

## Done when

- Import one real chat/export.
- Produce proposed lessons/open loops with source links.
- User can accept/reject/defer in Curation.
- Accepted item updates Standards/Practices/Routines only through the normal Curation path.
- Rejected item is remembered as rejected for that source, not silently retried forever.
