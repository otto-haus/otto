# otto harness skills

Agent skills for Cursor/Codex working in the otto repo. Installed via [skills.sh](https://skills.sh) (`npx skills add`).

**Not the same as** `skill/` at repo root — those are Letta charter/runtime skills (discord, github, routine, …).

Full inventory, vet status, gates, and usage: [`docs/skills-installed.md`](../docs/skills-installed.md).

Quick invoke: `Read .agents/skills/<name>/SKILL.md and follow it.`
