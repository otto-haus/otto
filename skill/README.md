# Letta / charter skills

Charter and routine skills for the Letta runtime. Each subdirectory is one skill (`SKILL.md`).

**Harness skills** (Cursor/Codex, skills.sh) live separately under [`.agents/skills/`](../.agents/skills/README.md). See [docs/skills-installed.md](../docs/skills-installed.md).

| Path | Purpose |
|------|---------|
| `SKILL.md` | Charter operating contract |
| `routine/` | Routine skill |
| `discord/` | Discord outbound (red gate) |
| `github/` | GitHub operations |
| `browser-proof/` | Browser verification |
| `cognee/` | Cognee memory |
| `pdf/` | PDF handling |
| `1password/` | 1Password reads |
